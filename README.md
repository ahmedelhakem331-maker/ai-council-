# 🏛️ AI Council

**Five Minds. One Vision.**

A platform where 5 AI agents simulate a council discussion to produce the best possible output for any idea.

## 🚀 Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## 🤖 Agents

| Agent | Role |
|-------|------|
| 💡 Nova | Idea Generator |
| 🎨 Pixel | Creative Enhancer |
| 🔍 Cipher | Critical Analyst |
| 📊 Vector | Business Strategist |
| 👑 Apex | Chairman (Final Verdict) |

## 📄 Pages

- **/** — Landing page with features & pricing
- **/dashboard** — Submit ideas and run council sessions
- **/history** — Review past sessions

## 🔧 Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- localStorage for persistence
- Simulated AI responses (swap for real LLMs)

## 📦 Deploy

```bash
npm run build
npx vercel --prod
```
