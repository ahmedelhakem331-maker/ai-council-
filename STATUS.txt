🚀 TO RUN THE PROJECT:

1️⃣  BACKEND:
   cd backend
   python -m venv venv && source venv/bin/activate
   pip install -r requirements.txt
   npx prisma generate && npx prisma migrate dev --name init
   python -m uvicorn app.main:app --reload

2️⃣  FRONTEND:
   npm install && npm run dev

3️⃣  BROWSER:
   http://localhost:3000
